# Contributing to Astronomic (VSC)

Thank you for your interest in contributing to the **Astronomic** theme for Visual Studio Code! I really appreciate your help in advance.

## How to Contribute

We primarily use GitHub Issues to track bugs and feature requests. Before opening a new issue, please check the [existing issues](https://github.com/mystery-git/astronomic_theme_vsc/issues) to see if it has already been reported.

### Reporting Issues

If you spot any unthemed UI elements, out-of-place colors, or bugs, please use the **Bug report** template.

- **Provide a clear description** of the UI area that is missing a themed color.
- **Suggest a themed color** (preferably in HEX format) if you have one in mind.
- **Include screenshots** to help identify the target area quickly.
- **Fill in platform details**, such as your editor (VS Code, Cursor, etc.) and the extension version.

### Feature & Language Support Requests

If you would like to see dedicated theming support for a specific programming language, please use the **Language support request** template to open a new request.

- **Specify the language** (e.g., Python, Rust, JavaScript).
- **Detail the scale of work**, including which syntax items should change and your suggested HEX color mappings.
- **Provide screenshots** of the current vs. desired theming if possible.

## IMPORTANT Note for Other Editors

This repository is strictly for the **Visual Studio Code** (and VSC-based editors) version of the Astronomic theme.

**Issues or feature requests for other editors (e.g. JetBrains, Sublime Text, Vim/Neovim, etc.) will be closed** to keep the work clean and focused.

Other editors/IDEs will receive their own dedicated repositories TBA.
