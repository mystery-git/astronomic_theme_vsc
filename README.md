## Status: Pre-release

![Screenshot of incomplete theme as of March 2026](/assets/images/theme_screenshot.png)

**Astronomic** is a bold color theme designed after the official colors of Astral (the Python tooling company). Aiming to be as **faithful to the color scheme** as possible.

Rock it if your whole set up is Astral or you just really like the colors!

---

***This theme is currently under development so it may undergo major changes before the final stable version.***

---

### Installation

Before this theme is made available in the VS Code marketplace and others you can choose to install it manually by:

- Finding your VSC extensions folder. Depending on your operating system you may find it in:
    - **Windows**: `%USERPROFILE%\.vscode\extensions`
    - **macOS**: `~/.vscode/extensions`
    - **Linux**: `~/.vscode/extensions`
- Copying the entire contents of the repository into your VSC extensions folder. 
- Restarting VSC before you can use it.
